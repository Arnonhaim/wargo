# WARGO — Landing Page + Admin CMS

דף נחיתה + מערכת ניהול תוכן לאפליקציית WARGO — פלטפורמת הובלות חכמה לישראל.

## 🚀 Deploy to GitHub Pages

1. צור repository חדש ב-GitHub
2. העלה את הקבצים מתוך ה-ZIP
3. לך ל-**Settings** → **Pages**
4. בחר **Source: Deploy from a branch**
5. בחר את ה-branch `main` והתיקייה `/ (root)`
6. לחץ **Save**

האתר יהיה זמין תוך דקה בכתובת:
`https://<your-username>.github.io/<repo-name>/`

## 🔐 מערכת ניהול (Admin CMS)

גישה לפאנל הניהול:
```
URL:  https://<your-site>/admin/
User: admin
Pass: Wargo2026!
```

מה ניתן לערוך:
- **Hero** — כותרת, תיאור, כפתורים, סטטיסטיקות, מוקאפ טלפון
- **יתרונות** — הוספה, מחיקה, עריכה וסידור מחדש של כרטיסי יתרונות
- **איך זה עובד** — עריכת השלבים
- **קהלי יעד** — כרטיסי שולחים ומובילים
- **סוגי רכבים** — עריכת רשימת כלי הרכב
- **קישורים** — App Store, Google Play, קישורים משפטיים
- **מיתוג** — צבעים, לוגו, שם האפליקציה
- **Footer** — תוכן ותיאור
- **SEO** — כותרת, תיאור, מילות מפתח, OG tags

## 📁 מבנה

```
├── index.html          ← דף הנחיתה
├── admin/
│   └── index.html      ← מערכת ניהול (CMS)
└── README.md           ← הקובץ הזה
```

## 🎨 התאמות

- **צבעים**: משתני CSS ב-`:root` בתחילת הקובץ
- **תוכן**: כל הטקסט בעברית ב-HTML
- **קישורים משפטיים**: עדכן את הקישורים בפוטר (תנאי שימוש, מדיניות פרטיות)
- **מייל תמיכה**: עדכן את `support@wargo.co.il` למייל שלך

## ✨ פיצ׳רים

- עיצוב מודרני ומקצועי בהתאמה לברנדינג של האפליקציה
- RTL מלא (עברית)
- רספונסיבי — מותאם למובייל, טאבלט ודסקטופ
- אנימציות scroll חלקות
- מוקאפ טלפון אינטראקטיבי
- כפתורי הורדה App Store / Google Play
- קובץ HTML יחיד — אין תלות חיצונית (מלבד Google Fonts)
